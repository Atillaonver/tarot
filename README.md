## What's used

* html5 boilerplate
* gulp
* gulp-sass
* browser-sync
* wiredep injection
* FontAwesome
* Twitter Bootstrap
* jQuery
* npm
* bower

## How to serve?

There are number of dependencies that are required to set up the development environment, install them using

```
npm install
bower install
```

To hookup browser-reload and start serving on localhost just type:

```
gulp
```
